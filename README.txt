care-taxi-shell-v14-safe-area-lock

目的：
レイヤー説明PDFの必要部分だけをトレースし、LOCATION / DETAIL / DAYS / STATUS の空間を固定する版。

重要：
・赤線、破線、枠線は本番表示しない。
・上部のiPadステータス文字（2:36、Wi-Fi、80%など）は表示しない。
・青帯、日付、時刻、予約登録ボタン、4エリアのラベルだけ表示する。
・各エリアの座標は、後から中身を入れるための透明レイヤーとして固定する。
・左右の縦基準は、揃え線であり安全限界線でもある。ここから外へ本文要素を置かない。
・PCでもiPadでも、1024×768のキャンバス全体を中央に縮尺表示する。
・画面全体スクロールは禁止。

次工程：
LOCATIONの中に、担当者列・時間軸・30分線・横スクロール層を入れる。
