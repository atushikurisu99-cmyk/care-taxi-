目的：
レイヤー説明.pdf を基準に、LOCATION / DETAIL / DAYS / STATUS の4エリアだけを先に固定する。
今回は中身を作り込まない。

重要：
- iPad Safariで真っ白にならないよう、import/exportやビルド前提を使わない。
- index.htmlを直接開ける構成。
- 白キャンバスは1024×768固定。
- JSで画面サイズに合わせてキャンバス全体を縮尺表示。
- 各エリアは後からHTML/CSS/JSで中身を追加できる。

次の工程：
1. LOCATION内に担当者列・時間軸・30分線を入れる。
2. DETAILは選択内容表示だけを入れる。
3. DAYSは7日カードを入れる。
4. STATUSはLOCATIONと同じ並びで状態表示を入れる。

設計ルール：
- 画面全体スクロールは禁止。
- 各エリアの位置を先に固定し、中身は後から追加。
- レイヤー境界を崩さない。
- PC表示ではブラウザ全体ではなく、白キャンバス内の比率を優先する。
