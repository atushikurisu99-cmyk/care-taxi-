Status Manager PC Layout v1

目的：
PDF「ステマネ02(自PC02)」を基準に、まずPC画面の実寸レイアウトを再現する表示専用モック。
リンク・保存・Google連携・クリック動作は未実装。

設計思想：
1. 左メニューは固定幅寄り。
2. 右DETAILも固定幅寄り。
3. 中央業務エリアを主役にする。
4. 横比率の差分は、主に以下のガターで吸収する。
   - 左メニューと中央の隙間
   - 中央とDETAILの隙間
   - DETAIL右側の余白
5. パーツそのものを極端に伸縮させない。
6. 高さ方向は、一覧エリアを優先して調整する。

ファイル：
- index.html
- styles.css
- app.js
