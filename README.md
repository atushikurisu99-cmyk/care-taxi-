# care-taxi-shell-v15-pdf-measured

## 目的
PDFトレース前提の「骨格固定版」です。
中身はまだ作り込んでいません。

## 表示対象
- 青帯
- LOCATION
- DETAIL
- DAYS
- STATUS

## 表示しないもの
- 赤破線
- オレンジ破線
- 黒線
- iPadのWi-Fi/バッテリー等の端末ステータス文字
- 枠線

## 方針
- PC表示は常に中央固定
- 背景は全面白
- iPad横向き基準
- 外枠座標は `css/layout.css` に集約
- 個別修正は各エリア内の中身だけ

## 確定前に見る場所
通常確認：
`index.html`

内部の透明エリア確認：
`index.html?debug=1`

※ debug は確認用。通常表示には線は出ません。

## 外枠座標
基準キャンバス：859 × 612

- 青帯: x=24, y=42, w=812, h=52
- LOCATION: x=48, y=108, w=592, h=334
- DETAIL: x=666, y=108, w=150, h=334
- DAYS: x=48, y=462, w=592, h=78
- STATUS: x=666, y=462, w=150, h=78

この数値は外枠固定用です。
中身の調整では原則変更しません。
