# care-taxi-shell-v3

介護タクシー管理アプリのShell骨格版です。

## この版でやること

- 青帯はブラウザ幅いっぱいに表示する
- 白本体は画面下端まで届かせる
- 全体スクロールは禁止する
- LOCATION / DETAIL / DAYS / STATUS の外接矩形だけを固定する
- 赤いガイド線・波線・薄い設計線は描画しない

## 重要な設計順序

1. Shellで大枠を固定
2. 4エリアの外接矩形を固定
3. その後、各エリアの中身を別ファイルで流し込む

LOCATION / DETAIL / DAYS / STATUS 側のコードは、自分の枠の外側位置を決めない。
位置・幅・高さは必ずShellが管理する。

## スクロール方針

このShell版では全体スクロールのみ禁止。
LOCATION内部スクロールやDETAIL内部スクロールは、各エリア実装時に個別に許可する。
