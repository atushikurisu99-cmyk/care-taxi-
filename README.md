# care-taxi-traced-v5

PDFの薄いグレー線で示された4つの外接矩形を、表示しない設計境界として固定した版です。

## 固定するエリア

- LOCATION
- DETAIL
- DAYS
- STATUS

この4エリアの境界は、UI部品を置くための基準であり、完成画面には薄いグレー線や赤破線は描きません。

## 青帯

青帯はブラウザ横幅いっぱいに伸ばします。中身はPDFのバランス幅に合わせ、横に間延びさせません。

## スクロール

- 画面全体: 禁止
- 青帯: 禁止
- 白本体: 禁止
- LOCATION: タイムライン内部だけ縦横スクロール可
- LOCATION: 縦横同時スクロール禁止。1操作につき片方向へロック
- 業者名列: 固定
- 時刻ヘッダ: 固定
- DETAIL: 中身だけ縦スクロール可
- DAYS: 禁止
- STATUS: 禁止

## 実装方針

レイアウトの修正は、まず `css/layout.css` の4エリア外接矩形だけを調整します。
中身の部品位置を動かして外接矩形のズレをごまかさないでください。


## v6 最小修正
- 画面全体スクロール禁止、タイムライン内スクロールの既存仕様は変更しない。
- LOCATION左の業者名セル部分を縦方向にドラッグした時だけ、タイムライン本体の縦スクロールへ同期する。
- 業者名セルでは横スクロールさせない。横スクロールは従来通りタイムライン本体側で行う。
