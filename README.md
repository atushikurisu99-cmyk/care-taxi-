# care-taxi-shell-v8

## この版の目的
添付PDFの必要部分だけをトレースし、以下の基準線を先に固定する。

- 青帯
- LOCATION
- DETAIL
- DAYS
- STATUS

## 重要思想
これは完成UIではなく、骨格の再現確認用。
データや装飾を作り込む前に、PDFの赤破線で示された基準線を崩さないことを優先する。

## 守ること
- iPadのOS表示、Safari表示、バッテリー表示はコードで描かない。
- 赤ガイド線は通常表示しない。
- 青帯は画面横いっぱい。
- 青帯内の要素は topInner 基準で固定。
- LOCATION / DETAIL / DAYS / STATUS は layoutFrame 基準で固定。
- 画面全体スクロールは禁止。
- LOCATIONの横スクロールだけ許可。
- 縦横同時スクロールは禁止。
- タイムラインの担当者列は固定。
- 時間線は6:30〜20:00まで内部的に持つ。
- 表示上の時間ラベルは7:00〜20:00。
- 30分線を基準に描画する。

## 確認用ガイド
URL末尾に `?guide=1` を付けると、確認用の赤破線を表示する。
例: `index.html?guide=1`

本番確認では `?guide=1` を外す。
